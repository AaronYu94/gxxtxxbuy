# 08 社群置顶规则执行单

> 目标：用 Discord / Reddit 置顶规则承接 first haul、QC、shipping、coupon 等高频问题，同时明确官方边界，避免用户误以为平台背书第三方商品。

## 社群目标

- 帮助新手快速理解首单流程。
- 把重复问题导向 First Haul Guide、Trust Center、Shipping FAQ。
- 明确 creator / community 分享不等于 GOATEDBUY 官方推荐。
- 给客服和社区运营统一处理口径。

## 推荐频道

| 频道 | 用途 | 置顶内容 |
| --- | --- | --- |
| `first-haul-help` | 新手首单帮助 | First Haul Guide、Paste Link 教程、费用说明。 |
| `qc-help` | QC 图片理解和下一步 | QC policy、补拍/退换说明、联系客服入口。 |
| `shipping-help` | 发货、运费、线路、体积重 | Shipping FAQ by country、estimate/final 说明。 |
| `haul-reviews` | 用户自愿分享 haul | 隐私提醒、不得发布敏感信息。 |
| `creator-codes` | creator code 和活动 | code 规则、免责声明、不可滥用规则。 |
| `support-updates` | 官方规则更新 | Trust Center 更新、已知问题、客服高峰提示。 |

## 置顶规则模板

```text
Welcome to GOATEDBUY community help.

Find your item anywhere. Paste the link on GOATEDBUY. We help with purchasing, warehouse handling, QC photos, shipping, and support.

Community and creator links are shared by third parties. GOATEDBUY does not officially recommend, verify, or endorse third-party items.

Start here:
1. First Haul Guide: [link]
2. Fees and shipping rules: [link]
3. QC policy: [link]
4. Refund policy: [link]
5. Contact support: [link]
```

## 官方回复边界

| 可以回复 | 不应回复 |
| --- | --- |
| 如何 paste link。 | 推荐具体商品或店铺。 |
| 如何查看 QC。 | 判断商品真伪。 |
| 如何理解运费和体积重。 | 承诺最终运费或到达日期。 |
| 如何申请补拍、退换、退款。 | 保证卖家一定同意退换。 |
| 如何使用 coupon。 | 鼓励过度消费或抽奖式刺激。 |

## 内容审核建议

| 风险 | 处理 |
| --- | --- |
| 用户发布地址、电话、订单号 | 提醒删除或由管理员隐藏。 |
| 违规商品或敏感内容 | 按社区规则删除并记录。 |
| 冒充官方背书 | 删除或标注第三方分享。 |
| creator 夸大承诺 | 要求修改，严重时暂停 code。 |
| 运费/退款错误信息 | 用官方规则链接纠正。 |

## 埋点和运营指标

| 指标 | 口径 |
| --- | --- |
| 社群回站率 | 社群链接点击到官网的用户数。 |
| First Haul Guide 点击 | 社群置顶到教程的点击。 |
| Shipping FAQ 点击 | shipping-help 到 FAQ 的点击。 |
| support 转化 | 社群到 support 的点击。 |
| 重复问题数量 | 同类问题周度数量。 |

## 验收标准

- 主要社群频道有置顶规则。
- 规则链接指向 First Haul Guide、Trust Center、Shipping FAQ。
- 免责声明明确第三方内容不等于平台背书。
- 社区运营和客服使用同一套规则链接。
- creator code 频道不泄露用户敏感数据。
