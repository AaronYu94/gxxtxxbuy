# 10 内容审核与合规 SOP

> 目标：统一处理 Haul Stories、社群内容、creator 内容和平台规则内容，降低违规商品、误导承诺、隐私泄露和平台背书风险。

## 审核范围

| 内容 | 审核重点 |
| --- | --- |
| Haul Stories | 隐私、违规商品、背书、敏感图片。 |
| Creator landing | 误导承诺、coupon 夸大、官方推荐暗示。 |
| 社群置顶 | 规则准确性、免责声明。 |
| 用户评论/分享 | 地址、订单号、tracking、违规内容。 |
| Trust/Country content | 费用、退款、物流规则准确性。 |

## 违规类型

| 类型 | 示例 |
| --- | --- |
| privacy_leak | 地址、电话、订单号、tracking。 |
| prohibited_item | 违规或敏感商品。 |
| misleading_claim | 保证真伪、保证到货、保证退款。 |
| official_endorsement | 暗示 GOATEDBUY 官方推荐商品。 |
| coupon_misuse | 夸大优惠、虚假折扣。 |
| harassment_spam | 垃圾信息或攻击性内容。 |

## 处理动作

| 动作 | 使用场景 |
| --- | --- |
| approve | 内容合规。 |
| request_edit | 内容可修改后发布。 |
| reject | 内容不适合发布。 |
| hide | 已发布内容临时隐藏。 |
| remove | 删除违规内容。 |
| escalate | 升级法务/风控/安全。 |
| suspend_creator | creator 严重违规。 |

## SLA 建议

| 风险等级 | 处理时间 |
| --- | --- |
| 高 | 2 小时内处理。 |
| 中 | 24 小时内处理。 |
| 低 | 3 个工作日内处理。 |

## 审核记录

必须记录：

- 内容 ID。
- 风险类型。
- 审核人。
- 操作动作。
- 操作原因。
- 时间。
- 是否升级。

## 验收标准

- Haul Stories 有审核状态。
- 用户可举报内容。
- 隐私泄露可快速隐藏。
- creator 违规承诺可追踪和处理。
- 审核动作进入审计日志。
