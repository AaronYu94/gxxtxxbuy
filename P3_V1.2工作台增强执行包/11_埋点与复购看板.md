# 11 埋点与复购看板

> 目标：衡量 P3 是否提升工作台资产沉淀、发货转化、复购和 creator campaign 效果。

## P3 事件表

| 编号 | 事件名 | 页面 | 时机 | 必填字段 |
| --- | --- | --- | --- | --- |
| C-001 | `dashboard_view` | Dashboard | 工作台访问 | `next_action_type`、`active_haul_count` |
| C-002 | `link_save` | Link Inbox | 保存链接 | `url_domain`、`source_platform`、`creator_code` |
| C-003 | `link_inbox_view` | Link Inbox | 页面访问 | `saved_count`、`parsed_count` |
| C-004 | `saved_link_add_to_haul` | Link Inbox | 加入 My Haul | `saved_link_id`、`parse_status` |
| C-005 | `haul_builder_view` | Haul Builder | 页面访问 | `ready_count`、`not_ready_count` |
| C-006 | `parcel_draft_create` | Haul Builder | 创建草稿 | `selected_count`、`estimated_weight` |
| C-007 | `shipping_preview_view` | Shipping | 预览发货 | `country`、`item_count`、`chargeable_weight` |
| C-008 | `shipping_preview_submit` | Shipping | 提交预览 | `line`、`estimated_fee`、`coupon_used` |
| C-009 | `savings_stack_view` | Wallet | 查看 savings | `total_saved`、`saving_count` |
| C-010 | `haul_history_view` | History | 查看历史 | `haul_count`、`delivered_count` |
| C-011 | `start_another_haul_click` | History | 点击复购 | `last_haul_status`、`country` |
| C-012 | `haul_recap_publish` | Recap | 发布 recap | `privacy_level`、`review_status` |
| C-013 | `haul_recap_share_click` | Recap | 分享 recap | `channel`、`recap_id` |
| C-014 | `creator_campaign_touch` | Creator | 记录触点 | `creator_id`、`campaign_id`、`source` |
| C-015 | `creator_repeat_conversion` | Creator | 复购归因 | `creator_id`、`campaign_id`、`repeat_count` |

## P3 看板

| 指标 | 口径 | 目标方向 |
| --- | --- | --- |
| Link save 率 | link_save / workspace UV | 提升 |
| Save to haul 转化 | saved_link_add_to_haul / link_save | 提升 |
| Haul Builder 使用率 | haul_builder_view / My Haul UV | 提升 |
| Draft to parcel 转化 | parcel_submit / parcel_draft_create | 提升 |
| Shipping Preview 支付转化 | shipping_pay_success / shipping_preview_view | 提升 |
| Savings Stack 使用率 | savings_stack_view / wallet UV | 可监控 |
| Haul History 使用率 | haul_history_view / completed haul users | 提升 |
| 30 日复购率 | 30 日内再次提交采购或发货用户 / 完成首单用户 | 提升 |
| 平均 haul 数 | haul 数 / 活跃用户 | 提升 |
| creator 复购转化 | creator_repeat_conversion / creator_first_order | 提升 |

## 归因分析

```text
Saved link
-> Add to haul
-> Build parcel
-> Shipping payment
-> Haul history
-> Start another haul
```

| 分析问题 | 需要事件 |
| --- | --- |
| Link Inbox 是否带来更多下单？ | `link_save`、`saved_link_add_to_haul`、`purchase_order_submit` |
| Haul Builder 是否提升发货？ | `haul_builder_view`、`parcel_draft_create`、`parcel_submit` |
| Shipping Preview 是否提升支付？ | `shipping_preview_view`、`shipping_pay_success` |
| History 是否促进复购？ | `haul_history_view`、`start_another_haul_click`、`purchase_order_submit` |
| creator 是否带来复购？ | `creator_first_order`、`creator_repeat_conversion` |

## 隐私要求

- 不上报完整 URL 参数、地址、支付信息、tracking number。
- user_id 用内部安全方式处理，面向 creator 必须脱敏或汇总。
- recap 分享事件不包含未公开内容字段。

## 验收标准

- P3 新事件在测试环境可见。
- 复购漏斗可计算。
- creator campaign 可区分首单和复购。
- 敏感信息不上报。
- 看板能按 source、creator、country、shipping line 维度切分。
