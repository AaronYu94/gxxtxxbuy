# Payment logo sources

These files are original brand artwork, not CSS recreations.

- `visa.svg`: Wikimedia Commons `Visa Inc. logo (2021-present).svg`; official brand guidance: https://corporate.visa.com/en/about-visa/brand.html
- `mastercard.svg`: Wikimedia Commons `Mastercard 2019 logo.svg`; artwork source: https://www.mastercard.us/content/brandcenter/standard-mastercard/en/download-artwork.html
- `discover.svg`: Wikimedia Commons `Discover Card logo.svg`; official media downloads: https://www.discover.com/company/newsroom/media-downloads.html
- `diners-club.svg`: Wikimedia Commons `Diners Club Logo3.svg`; official media downloads: https://www.discover.com/company/newsroom/media-downloads.html
- `unionpay.svg`: Wikimedia Commons `UnionPay logo (Full).svg`; official artwork center: https://www.unionpayintl.com/en/mediaCenter/brandCenter/artworkDownloadCenter/identification.shtml
- `jcb.svg`: Wikimedia Commons `JCB logo.svg`; official emblem agreement: https://www.global.jcb/en/about-us/brand-concept/agreement/

Only display payment marks for networks GOATEDBUY is contractually authorized to accept, and follow each brand's current usage rules.
